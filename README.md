#Casamento de Padrões - Algoritmos e Estruturas de Dados II

Este repositório contém o Trabalho Prático 03 da disciplina de Algoritmos e Estruturas de Dados II da Universidade Federal de Ouro Preto (UFOP). O projeto consiste na implementação e análise comparativa de desempenho de seis algoritmos clássicos para busca de padrões em strings.
